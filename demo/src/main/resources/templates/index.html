Welcome to my Spring web application show-case!

So... What's in there?

I've tried to incorporate the best technologies available at the moment... Here are some of them :

  • The project is built upon Spring MVC + Spring Boot for Java
  • The security and log in management is assured by Spring Security
  • Project building and dependencies management with Maven
  • For database access, I use Hibernate
  • Queries are made with Ajax to reload only a portion of the page and enhance performances
  • Thymeleaf handles pages layout and data binding
